# Social Media App

---

## Quick Start

Preview Link : https://connectier.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/prathmeshjagtap/social-media.git
```

Go to the project directory

```bash
  cd social-media
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features Available.

- Add Post
  - Add Post
  - Edit Post
  - Delete Post
  - Like Post
  - BookMark Post
  - Add Comment on Post
- Profile Feature
  - Edit Profile
  - Add Profile Picture
  - Add Bio
  - Add Website
- Follow Unfollow
  - Follow User
  - Unfollow User
  - visit user page
- Filters
  - Filter By Date
  - Filter By Trending Post
- Authentication
  - Login
  - Sigup
  - Guest Login
- Fully Resposive
- Dark Theme Available
